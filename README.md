# Membership System

A full-stack membership management system with Next.js frontend and NestJS backend.

## Project Structure

```
project/
├── frontend/     # Next.js + shadcn UI
└── backend/      # NestJS + Supabase + QuickBooks
```

## Frontend
- **Framework**: Next.js
- **UI Library**: shadcn
- **Port**: 3000

## Backend
- **Framework**: NestJS
- **Database**: Supabase
- **Integration**: QuickBooks
- **Port**: 8000
